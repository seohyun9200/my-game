# 4주차 실습 정리

## 오늘 한 것
- 원형 / AABB / OBB Bounding Box 시각화
- 세 방식의 충돌 판정 차이 관찰

## 관찰 내용

### 원형 vs AABB
원형과 AABB를 동시에 넣어서 관찰했다. 
원형 출동감지는 캐릭터 안쪽에 생성되어서 실제 충돌범위보다 적은 범위를 감지했고, AABB는 회전에 따라 범위가 바뀌어서 캐릭터 모양에 꽤 비슷한 범위를 감지했다.

### AABB vs OBB
AABB와 OBB를 동시에 넣어서 관찰했다. 캐릭터가 사각형 모양이라 OBB가 감지하는 범위가 정확하게 맞아떨어졌다. AABB는 계속 범위가 바뀌어서 OBB보다는 정확성이 떨어졌다.

## AI와의 대화에서 배운 것
생각보다 더 자세하게 말해야 내가 생각하는대로 만들어 준다는 것을 알게 되었다. 모르는 코드를 설명해주기도 했는데, 완전히 이해하기에는 힘들었다. 

## 내 게임에 적용한다면
캐릭터나 사물의 모양에 따라 사용하는 방식이 달라질 것 같다. 사각형 박스나 물체에는 당연히 OBB를 사용할 것 같고, 모양이 복잡해 질수록 AABB와 원형을 사용할 것 같다.
